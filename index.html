    About Taekwondo Academy System

    Technical specifications and information

    Taekwondo Academy Manager v1.0.0

    A state-of-the-art Single Page Application (SPA) designed to enable Taekwondo Academy operators to manage memberships, attendance sheets, and financial margins offline and online. Features instant responsive tracking and fully conforms to PWA standards for homescreen installation and lightning-fast loading speeds.

    Features Highlight

    SPA Routing Frictionless hash routing ensures views swap instantly with custom visual transitions and zero browser reload lag.
    Firebase Backend Syncs automatically with Cloud Firestore and Firebase Authentication if config is present.
    Smart Fallback No database credentials? No problem! The system shifts database calls to LocalStorage dynamically.
    Mobile PWA Fully installable on smartphones and tablets. Offers offline database queries and cached asset assets.

    Connecting Your Firebase Project

    To connect this application to your personal remote database, update the configuration block inside:

    // file: js/firebase-config.js
    const firebaseConfig = {
      apiKey: "YOUR_FIREBASE_API_KEY",
      authDomain: "YOUR_PROJECT_ID.firebaseapp.com",
      projectId: "YOUR_PROJECT_ID",
      ...
    };
